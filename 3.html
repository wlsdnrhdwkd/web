javascript

종종 JS로 축약되는 JavaScript (/ ˈdʒɑːvəˌskrɪpt / [6])는 고수준, 동적, 약한 유형, 프로토 타입 기반, 다중 패러다임 및 해석 된 프로그래밍 언어입니다. HTML 및 CSS와 함께 JavaScript는 World Wide Web 콘텐츠 제작의 세 가지 핵심 기술 중 하나입니다. 웹 페이지를 대화 형으로 만들고 비디오 게임을 포함한 온라인 프로그램을 제공하는 데 사용됩니다. 대부분의 웹 사이트에서이를 사용하며 모든 최신 웹 브라우저는 내장 JavaScript 엔진을 통해 플러그인 없이도이를 지원합니다. 많은 JavaScript 엔진 각각은 ECMAScript 사양을 기반으로 한 JavaScript의 다른 구현을 나타냅니다. 일부 엔진은 사양을 완전히 지원하지 않고 많은 엔진이 ECMA 이상의 추가 기능을 지원합니다.
